横須賀市役所 内部ツール

部署タイプ診断
その仕組み

5つの軸 × 20問で職員のタイプを判定し、
全103部署を適合度順にランクします。
入力 / Input

5つの軸 × 20の質問

A
人との関わり方
制度・仕組み ⇄ 市民対話
B
仕事の進め方
政策立案 ⇄ 現場対応
C
担う役割
ルール管理 ⇄ 市民支援
D
変化への姿勢
革新推進 ⇄ 安定運営
E
知識のスタイル
専門追求 ⇄ 幅広対応
質問の例：A1

高齢の市民が窓口を訪れ、介護申請の手続きに困っている。じっくり話を聞きながら、一緒に書類を進めていく。

自分には向いていないと思う
あまり気が乗らないが、こなせる
どちらとも言えない
やりがいを感じながら取り組める
まさに自分が輝ける場面
全20問・約3分
採点 / Scoring

回答が軸スコアに変わる

A1窓口で介護申請を支援+2
A2地域サークルで対話+1
A3業務フローを設計−1
A4財政データを分析+0
↓ 平均
A軸スコア+0.5

仕組み

  • 1つの質問は、1つの軸に寄与
  • 5段階の回答を数値（−2, −1, 0, +1, +2）に変換
  • 「反転質問」は符号を反転してから集計
  • 各軸ごとに、寄与する質問の 平均 を取る
軸スコア ＝ 各軸の（反転補正済み回答値）の 平均 範囲：おおよそ −2 〜 +2
比較 / Comparison

5次元ベクトル vs 103部署

受検者の結果（例）

A+0.6
B−1.0
C+1.2
D+0.2
E+0.4

部署の理想ベクトル（例：地域福祉課）

A+2.0
B+0.5
C+2.0
D+0.5
E+0.5
距離 ＝ √Σ（受検者ᵢ − 部署ᵢ）²　　→　　適合度% ＝ （1 − 距離 / √80）× 100 距離が小さいほど、適合度が高い。最大距離 ＝ √80 ≈ 8.94
部署ベクトルは103部署それぞれに事前設計（職務内容に基づく）
結果 / Output

アーキタイプ ＋ 部署ランキング

受検者が見る結果

32のアーキタイプ（2⁵）から1つに分類され、全103部署が適合度順にランクされます。

※ MBTI（4軸 → 16タイプ）の発想を参考に、5軸 → 32タイプへ拡張。 職員の性格分類ではなく、業務適性の把握を目的としています。
まとめ / Summary

仕組みの全体像

01
入力：20問が5軸スコアに
5段階回答に反転補正をかけ、軸ごとに平均を取る
02
比較：5次元距離で全103部署を採点
受検者ベクトルと部署ベクトルのユークリッド距離から適合度を算出
03
出力：アーキタイプと部署ランキング
32アーキタイプから1つ、103部署を適合度順に提示
注記：本ツールは 配属判断ツールではなく、職員自身が業務適性を考えるための 自己理解の補助 です。配属の最終判断は人事部門が担います。
